# PVS-0
larp larp larp sahur

Binocular/Monocular head-mounted digital night vision capable of low light contrast perception. Because this isnt meant for any real duty, this just lets me wander the forest late at night. This is a unique night vision solution because instead of using expensive off-the-shelf monocular displays like most open-source NVG solutions using the v760 or v780 like the PVS69, I'm going to be taking a page out of the budget FPV playbook by putting an objective lens on a compact high-resolution screen in order to get my video feed.
Mounting solution is a Wilcox dovetail onto a L4 G24 bracket into a 3-hole Wilcox shroud.
Imaging device is an FPV low light/starlight camera.
